Important FAQ !!

What is the Document Object Model(DOM)?

DOM is a programming interface for web documents.It allows programming languages like JavaScript to access, manipulate, and interact with the content, structure, and styling of a webpage dynamically.

How do you select an element from the DOM?

To select elements from the DOM,because these allow you to target HTML elements by their tag name, ID, class, or other attributes. Once selected, you can manipulate the element or its content.

What is event delegation in the context of the DOM,and why is it useful?

Event delegation is a technique in JavaScript where you attach a single event listener to a parent element to handle events from its child elements.
It's useful cause it allows those in a position of power to assign tasks to others, which, in turn, will allow the delegator to focus on higher-end tasks and activities.

How do you manipulates an element's attributes and styles using the DOM?

The classList object only provide methods to change the class of an element. If you want to change other attributes like id, href, or src, you can use the setAttribute() method.
